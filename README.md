# Webinar Nexum AI — Landing

Landing de registro + página post-registro para el webinar "Todas las empresas necesitan IA. Casi nadie sabe cómo convertir eso en un negocio." (7 de mayo 2026, 14:00 UY).

## Estructura

- `index.html` — sitio standalone (React + Tailwind + Babel vía CDN). Es el entrypoint que sirve Vercel.
- `App.jsx` — componente React exportable (opcional, para portar a un proyecto Vite/Next con Tailwind + lucide-react).
- `assets/` — logo, favicon, fotos founders.

## Deploy

Hosted en Vercel. Push a `main` dispara el deploy automático.
